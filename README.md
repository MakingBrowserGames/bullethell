# bullethell
A bullet hell game in HTML5 and JS, using the p5.js library. You can play it [here](https://xithiox.github.io/bullethell/).

### Controls
* Arrow keys to move
* C to use a bomb
* P to pause
* X to use a slowdown
* Z to fire

#### Debugging controls
* F to toggle FPS display
* H to toggle displaying hitboxes
* S to toggle starfield
